# Kevin's Adventure Game

A text-based adventure game where you explore a world of mystery and danger as you navigate through forests, caves, villages, and mountains. Collect items, interact with characters, and uncover the secrets of this magical realm.

## Overview

This repository contains both the original version of the game and a refactored version with improved code structure and organization.

- **Original Version**: Located in the root directory
- **Refactored Version**: Located in the `src/` directory

## Refactored Version

The refactored version has been completely restructured to use an object-oriented approach with proper class hierarchies, improved error handling, and better code organization.

### Features

- Explore different locations: Village, Forest, Cave, Mountain
- Collect and use items
- Interact with NPCs and mythical creatures
- Random events and encounters
- Weather system
- Save and load game progress

### How to Play

1. Navigate to the `src/` directory
2. Run `python main.py` to start the game
3. Follow the on-screen instructions
4. Type `help` at any time to see available commands

See the [src/README.md](src/README.md) file for more details on the refactored version.

## Original Version

The original version is a simpler implementation using dictionaries for data structures.

### How to Play

1. Run `python main.py` in the root directory to start the game
2. Follow the on-screen instructions
3. Type `help` at any time to see available commands
